## 🎯 Project Overview
An end-to-end sales analysis project for **Misuo** built entirely in **Microsoft Excel**. The goal was to build a robust tool that allows stakeholders to monitor business health through automated KPIs and visual trends.

## 🚀 Technical Steps & Features
* **Data Cleaning (Power Query):** Used Power Query to clean, transform, and shape raw data (handling duplicates and formatting dates).
* **Data Modeling:** Established relationships between data tables to ensure accurate calculations.
* **Analysis (Pivot Tables):** Created complex Pivot Tables to aggregate sales by region, category, and time.
* **Interactive Dashboard:** Built using **Slicers** and **Timelines**, allowing users to filter data dynamically.
* **Custom KPIs:** Tracked Total Revenue, Units Sold, and Average Order Value (AOV).

## 🛠️ Skills Demonstrated
* Advanced Excel Formulas.
* Power Query (ETL).
* Data Visualization & Dashboard Design.

## 💡 Key Insights
* Identified a clear trend in top-performing product categories.
* Visualized regional sales distribution to help target future marketing efforts.
